SBC-10001

Last dates

These dates are not mandatory. The aim is to give you give further control (of what?)within overall contract start and end dates.

What is the last date for new starts?

The last date for new starts is the last possible date for a start milestone to be paid.
For example, 27 3 2024.

What is the last date for mid-course payments?

The last date for mid-course payments is the last possible date for a mid-course milestone to be paid.
For example, 27 3 2024.

What is the last date for completions?

The last date for completions is the last possible date for a completion milestone to be paid.
For example, 27 3 2024.

What is the last date for outcomes?

The last date for outcomes is the last possible date for an outcome milestone to be paid.
For example, 27 3 2024.

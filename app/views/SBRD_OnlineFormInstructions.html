Add a contract

Overview

In order to add a contract you must have the following details:

  • the contract number in the correct format
  • the Skills Bootcamp name
  • the provider's UKPRN
  • the contract start date
  • the contract end date
  • the unit cost
  • the calculation method
  • the procurement wave
  • if applicable, the last dates for start payments, mid-course payments, completion payments and outcome payments
  • if applicable, the percentages for start payments, mid-course payments, completion payments and outcome payments
  • if applicable, the qualifying period (in days) for start payments

Details will only be saved once you have added all the data and selected to save.

Submit learner data

This service is for organisations submitting, validating or viewing data for the following data collections:

  • Individualised learner records (ILR)
  • Earnings adjustment statements (EAS)
  • ESF (European Social Fund) supplementary data
  • Funding claims
  • Further education staff and vacancy data
  • Further education governor data
  • Skills Bootcamp contracts (SBC)
  • Search for employer identifiers using the Employer Data Service

Devolved Authorities can download reports and upload reference data.

Before you start

You will need login details for DfE Sign-in.

2025 to 2026 ILR submission reports

Required Report Description
16-19 Funding Calculation Full Output Report For funding model 25 (16-19), this report details outputs from the funding calculation.
16-19 Funding Claim Report For funding model 25 (16-19), this report outputs learner counts and funding in the format of the funding claim form.
16-19 Summary of Funding by Student Report For funding model 25 (16-19), this report outputs learner counts and funding in the format of the funding claim form.
Adult Funding Claim Report For funding models 35 (Adult skills), 38 (Adult Skills Fund) and 99 (Non-funded), this report displays funding in the format of the funding claim form.
Adult Learner Loans Bursary Occupancy Report For funding model 99 (Non-funded), this report details funding that is financed by an Advanced Learner Loan.
Apprenticeship Data Match Report For funding model 36 (Apprenticeships), this report details rule violations produced by the data lock process.
Apprenticeship Indicative Earnings Report For funding model 36 (Apprenticeships), this report details learners, learning delivery and funding.
Community Learning Summary of Learners by Non-Single Budget Category Report For funding model 10 (Community Learning), this report details counts of learners.
Devolved Adult Education Funding Summary Report For funding models 35 (Adult skills) and 38 (Adult skills fund), this report details funding where the source of funding is a devolved organisation.
Devolved Adult Education Occupancy Report For funding models 35 (Adult skills) and 38 (Adult skills fund), this report details learners, learning delivery and funding where the source of funding is a devolved organisation.
Devolved Skills Bootcamp Funding Summary Report For funding models 37 (Skills Bootcamps), this report details funding where a contract number for a devolved contract is recorded in the ILR.
Devolved Skills Bootcamp Occupancy Report For funding models 37 (Skills Bootcamps), this report details learners, learning delivery and funding where a devolved contract number is recorded in the ILR.
Funding Summary Report For funding models 35 (Adult skills), 36 (Apprenticeships), 37 (Skills Bootcamps), 38 (Adult skills fund), 99 (Non-funded) or 81 (Other adult), this report details funding by funding line type.
High Needs Student Detail Report For funding model 25 (16-19), this report details the high needs status of learners attracting 16-19 funding.
High Needs Student Summary Report For funding model 25 (16-19), this report details counts of learners attracting 16-19 funding, by funding line type.
Main Occupancy Report For funding models 35 (Adult skills) and 38 (Adult skills fund), this report details learners, learning delivery and funding.
Maths and English Report For funding model 25 (16-19), this report details the GCSE status for both Maths and English for each student attracting 16-19 funding.
Non-Contracted Apprenticeships Activity Report For funding model 36 (Apprenticeships), this report details learning delivery that has been successfully validated, and therefore had funding calculated, but for which the provider does not have a valid contract
Non-Contracted Devolved Adult Education Occupancy Report For funding models 10 (Community Learning), 11 (Tailored Learning), 35 (Adult skills) and 38 (Adult skills fund), this report details learning delivery that has a source of funding for a devolved organisation and there is no associated contract.
OfS HE Quality Control Report Where higher education (HE) data has been returned, this report assists in monitoring and improving the quality of the HE data.
Project S Skills Bootcamp Occupancy Report For funding model 81 (Other adult), this report details Skills Bootcamp data where the learning delivery is recoded as 'Project S'.
Skills Bootcamp Funding Summary Report For funding model 37 (Skills Bootcamps), this report details Skills Bootcamp funding by contract number.
Skills Bootcamp Occupancy Report For funding model 37 (Skills Bootcamps), this report details learners, learning delivery and funding.
Summary of Funding Model 35 Funding Report For funding model 35 (Adult skills), this report details funding by funding line type and period.
Summary of Funding Model 35 Funding Report For funding model 35 (Adult skills), this report details funding by funding line type and period.
Trailblazer Apprenticeships Employer Incentives Report For funding model 81 (Other adult), this report details funding at an employer level for Trailblazer Apprenticeships.
Trailblazer Apprenticeships Occupancy Report For funding model 81 (Other adult), this report details learning, learning delivery and funding data for Trailblazer Apprenticeships.

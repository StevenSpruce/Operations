Payment Details

Qualifying Period

The start payment is based on a 5 day qualifying period (and other conditions before they are recorded in the ILR)
Start Payment
The start payment is based on evidence that the learner has an HGV provisional licence (and other conditions before they are recorded in the ILR)
%

What is the last date for start payment?
This is the last date we will generate a start payment for any bootcamps within this contract. This is the date by which the learner's activity should take place, not the date by which it should be recorded in the ILR. If a learner meets the criteria for the start milestone based on their activity after this date, we will not generate a start payment.

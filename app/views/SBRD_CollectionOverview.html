This collection allows you to submit the Skills Bootcamp contract reference data required to support the funding calculation.

Data can be entered using the online form service or imported using the CSV file upload service:

  • the online form service allows you to add contract reference data individually
  • the CSV file upload service allows you to bulk upload contract reference data

Skills Bootcamp reference data collection

This collection allows you to submit the Skills Bootcamp contract reference data required to support the funding calculation.

Data can be entered using the online form service or imported by using the CSV file upload service.

The online form service allows you to add contract reference data individually.

The CSV file upload service allows you to bulk upload contract reference data. Note, this service overwrites any current data with the content of the new file.

Additionally, this collection allows you to export your contract reference data as a CSV file.

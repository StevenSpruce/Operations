Guidance on how to complete the contract data CSV file

There are 10 data fields to fill out for each course.

Skills Bootcamp name

Column A

Data entry
Enter the name of the Skills Bootcamp.
Additional guidance
100 character max length
Example
Name of Skills Bootcamp

Provider UKPRN

Column B

Data entry
Enter the UKPRN of the bootcamp provider.
Additional guidance
8 character integer
Example
Provider UKPRN
10003467
10005478

Your contract reference number

Column C

Data entry
Enter the your contract reference number (optional).
Additional guidance
This can be up to 50 characters. It is an optional field
Example
Your Reference Number
(optional)
SBHGV12091
SBHGV1A132

Contract funding wave

Column D

Data entry
Enter the contract funding wave.
Additional guidance
Enter Skills Bootcamp funding wave does the contract belongs to.
Example
Contract wave
Wave 5

Calculation method

Column E

Data entry
Enter the calculation method relevant to the Skills Bootcamp contract
Additional guidance
Every learner should be linked to a Skills Bootcamp course.
Example
Contract Reference Number associated with learner
(Up to 20 characters starting with SB / SBA / SBD / XSB)
DSBC03
DHGV02

Contract start date

Column F

Data entry

Enter the contract start date

Additional guidance
This is the start date of the contract. No learning activity is expected before this date. No payments will be generated before this date.
Example
Contract start date
26/12/2024
17/11/2024

Contract end date

Column G

Data entry

Enter the contract end date

Additional guidance
This is the end date of contract. No payments will be generated after this date. (Optional field).
Example
Contract end date
(Optional)
03/03/2025
26/01/2025

Unit cost

Column G

Data entry

Enter the unit cost

Additional guidance
The unit cost is the overall agreed price of the Skills Bootcamp, before any reduction for employer co-funding.
Example
Unit cost
10000.00
2500.57

Qualifying period

Column I

Data entry

Enter the qualifying period of the contract

Additional guidance
This field only needs to be return if it is relevant to the calculation method that has been input for that contract. If a qualifying period isn't relevant, then the field should be left blank
Example
Qualifying period(Optional)
10
3

Start payment percentage

Column J

Data entry

Enter the start payment percentage

Additional guidance
This field only needs to be return if it is relevant to the calculation method that has been input for that contract. If the start payment percentage isn't relevant, then the field should be left blank
Example
Start payment percentage(Optional)
10
50

Last date for start payment

Column K

Data entry

Additional guidance
This field is optional and only needs to be return if needed by the contract owner and relevant to the calculation method
Example
Last date for start payment(Optional)
21/02/2024
12/04/2024
